Kelas HTML chapter 1

di chapter kali ini kita hanya akan mempelajari tag dari HTML. apa itu tag HTML, tag HTML adalah sebuah perintah untuk menuliskan sesuatu pada website, perlu di ingat bahwa HTML bukan bahasa pemrograman untuk styling, tapi hanya dalam penulisan pada sebuah website.

mengenal tag dasar HTML

  • h1 : untuk header
  • h2 : untuk subheader
  • h3 : untuk bab
  • h4 : untuk subbab
  • h5 : untuk lebih kecil dari sebelumnya
text di atas jika di tulis di HTML akan menjadi seperti ini

    ini tag h1

    ini tag h2

    ini tag h3

    ini tag h4

    ini tag h5
jangan lupa untuk mencobanya di teks editor kalian masing masing!

mengenal tag dasar HTML

  • p : untuk paragraf
  • i : untuk italic (huruf miring)
  • ul : untuk membuat daftar list
  • li : untuk membuat list
  • a : untuk membuat tombol
text di atas jika di tulis di HTML akan menjadi seperti ini

    ini paragraf

    ini huruf miring
      ul tidak terlihat
  • ini list
    • ini tombol
catatan!, jika kamu ingin membuat list sebaiknya tambahkan tag ul sebelum tag li. jangan lupa untuk mencobanya di teks editor kalian masing masing!
